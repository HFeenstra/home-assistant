# Home-Assistant
Current Version: 0.28.2
Running on a Raspberry PI 1 Model B

# Samba share
* ```sudo apt-get install samba```
* ```edit sudo nano /etc/samba/smb.conf```
* ```sudo smbpasswd -a pi```
* ```sudo service smbd restart```
